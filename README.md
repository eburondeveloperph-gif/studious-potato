# Eburon Client Apps

Deployment platform for applications generated by [Eburon Codepilot](https://codemax-cli.vercel.app).

## Structure

```
client/
  deployments/
    {user_id}/
      {app_name}/
        {timestamp}/
          index.html
          ...
```

## How it works

1. User generates an app via Eburon Codepilot chat
2. Generated files are auto-committed to this repo under `client/deployments/`
3. Vercel deploys the app instantly with a unique URL
4. The URL is shown in the Codepilot UI

## Development

```bash
npm install
npm run dev
```

## Deployment

This repo is connected to Vercel for automatic deployments on push to `main`.
